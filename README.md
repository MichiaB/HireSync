# HireSync

HireSync is a lightweight new hire tracker page with:
- Five active tabs: Homepage, Calendar, New Hires, Data, Settings
- Clickable new-hire profile cards
- Editable popup profile form with:
  - Name
  - Employee Number
  - Department
  - RAID Color
  - Status
  - Comment

## Run locally
Because this project is dependency-free (plain HTML/CSS/JS), open `index.html` directly in your browser.

Optional local server (if Python is installed):
```powershell
python -m http.server 5500
```
Then visit `http://localhost:5500`.
